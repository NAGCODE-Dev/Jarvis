# Official Documentation Registry

## Executive Summary

Use these sources first when answering programming questions in Jarvis. They are more stable and more reliable than random blog summaries.

## Registry

- HTTP and web semantics: https://www.rfc-editor.org/info/rfc9110/
- Python docs: https://www.python.org/doc/
- Python tutorial: https://docs.python.org/3/tutorial/index.html
- Git docs: https://git-scm.com/docs
- Pro Git: https://git-scm.com/book/en/v2
- Docker docs: https://docs.docker.com/
- Android docs: https://developer.android.com/develop
- Android component fundamentals: https://developer.android.com/guide/components/fundamentals
- Kotlin docs: https://kotlinlang.org/docs/home.html
- PostgreSQL docs/site: https://www.postgresql.org/
- OWASP Top 10: https://owasp.org/www-project-top-ten/
