# Programming FAQ

## Should I start with microservices?

Usually no. A modular monolith is often the better starting point.

## Are design patterns required in every project?

No. They are useful when they simplify recurring problems.

## Is Docker enough for production readiness?

No. You still need observability, security, deployment discipline, and failure planning.

## Is RAG better than fine-tuning?

Not universally. RAG is usually the better first move when the problem is external knowledge freshness and provenance.
