Jarvis Local não conseguiu iniciar automaticamente

O launcher Debian foi aberto, mas o core local não ficou saudável a tempo. O sistema continua recuperável em modo local.

Passo 1: rodar o diagnóstico

/opt/jarvis-local/app/scripts/jarvis.sh setup-local --dry-run
/opt/jarvis-local/app/scripts/jarvis.sh setup-local
/opt/jarvis-local/app/scripts/jarvis.sh doctor

Passo 2: preparar o runtime local

/opt/jarvis-local/app/scripts/jarvis.sh setup-local --dry-run
/opt/jarvis-local/app/scripts/jarvis.sh setup-local
# opcional:
/opt/jarvis-local/app/scripts/jarvis.sh setup-local --with-infra --with-models

Passo 3: tentar subir manualmente

/opt/jarvis-local/app/scripts/jarvis.sh boot --no-seed
/opt/jarvis-local/app/scripts/jarvis.sh status

Se faltarem dependências Python

/opt/jarvis-local/app/scripts/jarvis.sh setup-local
# ou, se quiser o setup completo:
/opt/jarvis-local/app/scripts/jarvis.sh setup-local --with-infra --with-models

Verificação final

/opt/jarvis-local/app/scripts/verify_local.sh
